A WebGPU backend behind a per-backend ESM loader

The browser loader no longer imports the monolith: it loads tfjs-core once, then dynamically imports exactly the one requested backend package — unblocking code-splitting and the WebGPU-vs-CPU race.

Comprehension job 1 · before vs built

The browser backend-loading path, before and after

src/backend/loader.browser.ts — the importer map that replaced the monolith import
// BEFORE — one hard import of the whole bundle (WebGL only):
const tf = await import('@tensorflow/tfjs');

// BUILT — core once, then dispatch to exactly one backend package:
const BROWSER_BACKEND_IMPORTERS: Partial<Record<TensorFlowBackend, () => Promise<unknown>>> = {
  cpu:    () => import('@tensorflow/tfjs-backend-cpu'),
  webgl:  () => import('@tensorflow/tfjs-backend-webgl'),
  webgpu: () => import('@tensorflow/tfjs-backend-webgpu'),
  wasm:   () => import('@tensorflow/tfjs-backend-wasm'),
};
const tf = await load_core();           // @tensorflow/tfjs-core
await BROWSER_BACKEND_IMPORTERS[requested]();  // registers that backend (+ cpu under a GPU lane)
Comprehension job 2 · the honesty crux + the script-tag hatch

WebGPU init: two distinct failure points, one real-lane guard

What to watch

The 'webgpu' backend is async-only / experimental: init and verification work, but the clustering algorithms read tensors back synchronously, so running fit / fit_predict on 'webgpu' is not yet supported (documented on Clustering.init). The race harness drives WebGPU with raw tensors, not the algorithms; making the algorithms webgpu-safe end-to-end is a separate follow-up.

Comprehension job 3 · the decision, resolved

Where does the per-backend loader map live? resolved → src/backend

Resolution

The per-backend map ships in src/backend/loader.browser.ts as a library feature. The BROWSER_BACKEND_IMPORTERS map is keyed by TensorFlowBackend with four entries — cpu, webgl, webgpu, wasm — and backend.ts early-returns this browser loader while node and react-native keep the generic flag / setBackend / ready tail.
