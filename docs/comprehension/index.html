Hover any box to trace its connections; click to open that explainer. Arrows point from a building block toward what consumes it.

How to read this atlas. Pick an estimator that matches your problem and read its explainer top-to-bottom — intuition, then the live demos, then the math. When a step mentions a building block (a distance matrix, an eigensolve, a Lance-Williams update, a seeded RNG), follow the link down into that numerics page to see exactly how it is computed. Finish with the tooling pages — Choosing k, Validation, and the deterministic RNG — which explain how results are scored, ranked, and made byte-for-byte reproducible across all four estimators.
